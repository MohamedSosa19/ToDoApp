# ✅ Todo Management App

Simple CRUD app using ASP.NET Core and JavaScript.

---

## 🚀 Features

- Add, edit, delete todos
- Filter by status
- Priority and due date support
- Bootstrap 5 UI

---

## ⚙️ Technologies

- ASP.NET Core 8 
- using 3-Tier Arch and using DTOS 
- Using Mapper 
- Entity Framework Core
- JavaScript (Vanilla)
- Bootstrap 5
